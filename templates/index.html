Портфолио

Я стремлюсь стать хорошим специалистом в сфере информационных технологий, поэтому постоянно развиваю свои компетенции. На данный момент я прохожу обучение по магистерской программе "Компьютерная лингвистика и искусственный интеллект" в Московском Политехе. В свободное время я выполняю работу над некоторыми проектами по программированию. Иногда я занимаюсь созданием 3д моделлей различных объектов.

Сертификат "Оператор ЭВМ"

Завершил обучение во Владимирском промышленно-коммерческом лицее. Углубленно изучал физику, математику и информатику. Успешно завершил изучение дисциплин связанных с программированием и работой с прикладным программным обеспечением. Получил навыки системного администрирования. Изучил базовые понятия в сфере 3д моделирования и обработки изображений с помощью редакторов.

Сертификат получен

Диплом Бакалавра

Завершил обучение в Московском политехническом университете по направлению прикладная информатика. Согласно программе профессиональной подготовки обладаю навыками осуществления всех этапов жизненного цикла программных продуктов. Подробно знакомился с шагом проектирования (моделирования) систем. Изучал программирование на 1С, Python, С#, а также получил навыки веб-разработки.

Диплом получен 06.07.2021

Бот Вконтакте

Проект заключается в создании чат бота для сообщества MovieNerds. Бот предназначен для автоматического взаимодействия с участниками группы или обычными пользователями социальной сети. Главной задачей является подбор кино/сериалов/аниме по заданным параметрам. Для получения информации о картинах бот взаимодействует с API сайта КиноПоиск.

Мини-приложение Вконтакте

Проект для личного пользования. Веб-приложение для Vk Mini Apps, которое позволит создавать список целей или задач. Для достижения поставленного результата предполагается система уведомлений и напоминаний. Завершение конкретной задачи, по желанию пользователя, формирует пост, которым можно поделиться.

Алгоритмы и задачи

Проект для развития навыков программирования и вёрстки. Репозиторий состоит из файлов-исходников, содержащих код реализации задач, а также таблицы отчётности. Вопросы программирования выбираю с Leetcode, а затем решаю их на языке Python. Задачи вёрстки состоят в формировании элементов/сайтов в соответствии с условиями Speedrun Challange

Игра платформер

Проект предназначенный для улучшнения навыков программирования на Python. Создаю простой 2д платформер с использованием библиотеки PyGame. Игра спроектирована в сеттинге постапокалипсиса и предполагает харакерный 8-битный дизайн окружения.
